# Teste Front-end #

## Instruções ##

No Visual Studio, em seu terminal entrar na pasta do projeto e digitar 'ng serve'.
Após rodar o projeto entrar no browser e digitar localhost:4200.

## Tecnologias ##

- Desenvolvimento em HTML, CSS, Javascript;
- Teste desenvolvido usando o Framework Angular 8;
- Os estilos da página foram desenvolvidos com o pré-processador SASS;
- Página responsiva utilizando Medias Queries em resolução para desktop e mobile (max-width: 767px);
